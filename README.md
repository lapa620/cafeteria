# cafeteria
Uma cafeteria temática de gatos (ou "Cat Café") pede um README que seja tão charmoso e acolhedor quanto o próprio estabelecimento.

Aqui está uma estrutura caprichada, organizada e pronta para você copiar e adaptar:

🐾 Cat Café - Página Inicial
Bem-vindo ao repositório da landing page do Cat Café! Este projeto é uma vitrine digital para uma cafeteria onde o café é quente, o ambiente é aconchegante e os anfitriões ronronam.

😺 Sobre o Projeto
A ideia deste site é transmitir a paz e a fofura de um café repleto de gatinhos. O design foca em uma experiência de usuário (UX) suave, com cores pastéis e elementos visuais que remetem ao universo felino.

✨ Funcionalidades
Menu Delicioso: Exibição de cafés, doces e petiscos (para humanos e felinos).

Galeria de Residentes: Conheça os gatinhos que vivem na casa e estão disponíveis para adoção.

Reserva de Horários: Formulário integrado para garantir seu lugar no "Playroom".

Newsletter "Miau": Inscrição para receber novidades e fotos de gatinhos toda semana.

🛠️ Tecnologias Utilizadas
Para construir este ambiente digital, utilizamos:

HTML5: Estrutura semântica.

CSS3: Estilização com Flexbox/Grid e animações suaves.

JavaScript: Interatividade (carrossel de fotos, validação de formulário).

Google Fonts: Fontes arredondadas e amigáveis (ex: Quicksand ou Fredoka).
